# Critic 22

A code review tool for Git repositories with TUI interface.

## Features

- Live diff viewer with syntax highlighting
- File watching for automatic updates
- Split pane interface
- Phase 2: Code review comments (coming soon)
- Phase 3: MCP server integration (coming soon)

## Usage

```bash
# View all changes
./critic

# View specific files
./critic path/to/file.go
```

## Keyboard Shortcuts

- `Tab`: Switch between panes
- `↑/↓` or `k/j`: Navigate
- `q`: Quit
