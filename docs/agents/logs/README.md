# Agent Task Logs

This directory contains progress logs for significant tasks worked on by AI agents.

## Purpose

- Track progress on complex tasks
- Document obstacles and their resolutions
- Capture learnings for future reference
- Provide visibility into agent work

## File Naming

Use the format: `YYYYMMDD-HHMMSS-short-description.md`

Examples:
- `20250115-143022-fix-scroll-crash.md`
- `20250116-091530-add-export-feature.md`
- `20250117-162845-refactor-event-loop.md`

## Quick Template

```markdown
# Task: [Brief description]

**Started:** YYYY-MM-DD HH:MM:SS
**Ended:** YYYY-MM-DD HH:MM:SS (or "in progress")
**Strategy:** [Bug Fix | Feature (TDD) | Refactoring | Performance]
**Status:** [In Progress | Completed | Blocked]

## Objective
[What we're trying to achieve]

## Progress
- [ ] Step 1
- [ ] Step 2

## Obstacles
[Document blockers and challenges]

## Outcome
[Final result when complete]
```
